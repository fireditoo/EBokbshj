# 项目简介

## 核心功能
- 基于YOLOv8的火焰实例分割检测系统
- 支持图片、视频及摄像头进行火焰检测
- 支持选择不同模型进行检测
- 界面实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并在界面直接查看所有检测结果
- 支持检测结果保存
- 支持自定义置信度和IOU阈值
- 可选择是否显示标签和原图

## 适用场景
- 火灾监测与预警系统
- 安全监控应用
- 工业环境安全监测
- 森林火灾预防
- 建筑安全检测

## 技术特点
- 采用Python 3.9开发
- 依赖环境包括opencv-python、PyQt5、ultralytics、torch1.9等
- 提供完整源码、训练好的模型及训练结果
- 内含400张标注数据集，包含-fire-一类
- 配有完整项目运行教程
- 提供UI界面源文件和图标资源
- 支持Pycharm+Anaconda或VSCode+Anaconda开发环境
